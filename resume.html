Linux from Scratch

Linux from scratch is project where you take all of the packages which make up a linux distribution, and then build it from the ground up. It took a month and a half working each night to construct my Linux distribution, called Archaea, which runs on the metal of my Thinkpad T420.
Archaea is LFS version 8.3, and is a minimal installation. You can find it in the Get Counted section of the LFS website under ID 27437, and you can find my compiled notes on Github. I have added to this LFS since constructing it: it is now a 'BLFS' with i3 as a window manager, wireless management, a modern web browser, a file browser and more.

SRControl

SRControl is a project that aims to bind Linux CGroups using python. It will allow users to execute specific functions in their own CGroups, as well as create and manage new CGroups generally.
SRControl is released under GPLv3 and can be found here. Feel free to issue pull requests!

LAD

This new project will allow users to enter any Linux Allocated Device major/minor combination and return the device name, and vice-verse ( e.g. /dev/sda2 -> 8:3 ).
This project is yet unlicensed. It will likely be BSD, or something. You can contribute here.

Grand Exchange Analysis

This project is written in python. A flask application communicates with a PostgreSQL instance in a docker container to grab item ids, and using this information, scrapes item price information from the Grand Exchange, the auction house of the online game RuneScape. Once the price information over the last 90 days is collected, the application runs one of three options in polynomial regression, returning an image of the regression curve overlaying the price values. The algorithm for regression is set up to run arbitrary degrees of regression. Here's a preview:
